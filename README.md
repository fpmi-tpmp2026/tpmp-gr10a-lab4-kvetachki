# Laboratory Work 4  
## Project Management, UML Specification, CI/CD, and C Application Development

## Overview
This repository contains the documentation and project management materials for Laboratory Work №4 in the course “Technologies of Programming for Mobile Applications”.

The laboratory work consists of two parts:

1. Individual assignment — development of a UML specification for the student’s own application (based on Task 2 from Laboratory Work №3).
2. Team assignment — development of a console application using Git, GitHub Actions, Kanban, UML, and SQLite.

This repository is used exclusively for documentation, UML diagrams, and project management.  
The source code is stored in a separate repository (see the link below).

The team uses GitHub Projects to manage tasks.
